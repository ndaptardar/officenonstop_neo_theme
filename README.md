# Officenonstop Neo Theme

Next-generation ERPNext v15 theme with four color schemes: **Light**, **Ocean**, **Sunset**, and **Forest**.  
Built with a modern design system using CSS variables and shadows for a polished SaaS look.

## Installation

1. Install app:
   bench --site your-site install-app officenonstop_neo_theme

2. Build assets (only if not prebuilt in Docker):
   bench build

3. Restart:
   bench restart

Each user's theme choice is persisted automatically in their ERPNext profile.
